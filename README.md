# hexirp-hakyll

hexirp-hakyll is a static web site generator library in Haskell **for Hexirp**.

hexirp-hakyll は static web site generator を構築するためのライブラリである。このライブラリは Haskell で書かれている。また、これは Hexirp のためのライブラリである。
